# 🚀 Quick Start Guide - Grow A Garden Script

## Langkah Cepat (3 Menit)

### 1. Download Script
```
Download semua file dari folder /root/GAG-farm-v1/
```

### 2. Upload ke GitHub
```
- Buka github.com
- Klik "New Repository"
- Nama: GAG-farm-v1
- Upload semua file
- Pastikan folder structure benar
```

### 3. Copy Loadstring
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/AUR4NK/GAG-farm-v1/main/main.lua"))()
```
**Ganti `AUR4NK` dengan username GitHub kamu!**

### 4. Jalankan
```
1. Buka Roblox
2. Masuk ke game "Grow A Garden"
3. Buka executor (Synapse/Fluxus/Delta)
4. Paste loadstring di atas
5. Klik Execute
6. Tekan RightShift untuk buka menu
```

## Fitur Cepat

### Auto Farm Coin 💰
```
Tab: Coin Farm
1. Toggle "Auto Farm" → ON
2. Toggle "Auto Sell" → ON
3. Set "Farm Mode" → all
4. Set "Sell Interval" → 30
5. Selesai! Script farm otomatis
```

### Cari Raccoon 🦝
```
Tab: Raccoon
1. Toggle "Auto Find Raccoon" → ON
2. Toggle "Auto Collect" → ON
3. Set "Scan Radius" → 500
4. Tunggu sampai ditemukan!
5. Jika menemukan spawn, klik "Save Current Position"
```

### Cari Unicorn 🦄
```
Tab: Unicorn
1. Toggle "Auto Find Unicorn" → ON
2. Toggle "Auto Hatch Eggs" → ON
3. Set "Scan Radius" → 600
4. Tunggu sampai ditemukan!
5. Script juga auto hatch egg legendary
```

### Auto Tame Pet 🐾
```
Tab: Tame Sniper
1. Toggle "Auto Snipe Tame" → ON
2. Toggle "Auto Server Hop" → ON
3. Set "Server Hop Mode" → low
4. Set "Max Tame/Server" → 5
5. Script cari & tame pet di server sepi
```

## Tips & Trik

### Performa
```
- Scan Radius: 300-500 (jangan terlalu besar)
- Scan Interval: 1000ms (balance speed & stability)
- Teleport Speed: 200-300ms (cepat tapi aman)
- Max Collect/Loop: 30-50 (jangan terlalu banyak)
```

### Anti-Detection
```
- Jangan terlalu cepat (set delay wajar)
- Jangan AFK terlalu lama (sesekali gerak)
- Gunakan mode "low" untuk server sepi
- Jangan collect terlalu banyak sekaligus
```

### Spawn Points
```
- Simpan posisi saat menemukan Raccoon/Unicorn
- Script akan otomatis cek posisi tersebut
- Bisa simpan banyak spawn point
- Spawn point tersimpan di session (hilang saat rejoin)
```

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Menu tidak muncul | Tekan RightShift |
| Script error | Pastikan executor support |
| Auto collect tidak jalan | Kurangi kecepatan |
| Pet tidak ditemukan | Tambah scan radius |
| Sell tidak jalan | Coba manual sell |
| Disconnect | Aktifkan Anti-AFK |
